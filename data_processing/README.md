# 数据处理介绍

## 目标
针对任务是要做800字小说创作，需要将17本小说进行处理成为可以为模型训练的数据

## 处理方法
1. 将17本小说按照不超过800字开始切分
2. 利用大模型对切分的小段进行总结（相当于生成小说的提示词）
3. 将提示词、总结和切分的原文组成一条数据
